# 组织社会学大富翁 🎮

一个以组织社会学为主题的互动教育游戏，采用类似大富翁的玩法。

## 🎯 游戏特色

- 📚 学习组织社会学的核心概念和理论
- 🎲 投掷骰子，在棋盘上移动
- 💡 回答问题获得积分
- 🎁 随机机会事件
- 🏆 达到100分即可通关
- 🔥 支持 Firebase 云端存储（可选）

## 🎮 游戏规则

1. **概念格子**（蓝色）：基础概念题，答对 +10 分
2. **理论格子**（橙色）：核心理论题，答对 +20 分
3. **机会格子**（粉色）：随机事件，可能获得或失去 10-15 分
4. 累计 100 分即可完成学习之旅

## 🔥 Firebase 云端存储配置（可选）

### 为什么配置 Firebase？

**不配置 Firebase**（默认）：
- ✅ 成绩保存在本地浏览器
- ❌ 其他学生看不到你的成绩
- ❌ 换设备后数据丢失

**配置 Firebase** 后：
- ✅ 所有学生成绩保存在云端
- ✅ 实时排行榜，看到全班成绩
- ✅ 老师可以在 Firebase Console 查看
- ✅ 数据永久保存

### 快速配置步骤

1. 访问 [Firebase Console](https://console.firebase.google.com/) 创建项目
2. 创建 Firestore 数据库
3. 在游戏中点击 "⚙️ 配置云端"
4. 填写 Firebase 配置信息
5. 完成！

详细步骤请查看 [FIREBASE_SETUP.md](./FIREBASE_SETUP.md)

## 📊 成绩记录

游戏支持多种成绩记录方式：

### 1. 本地存储（默认）
- 保存在浏览器中
- 只能看到自己的成绩
- 适合个人练习

### 2. Firebase 云端存储（可选）
- 保存在 Google 云端
- 所有人可见排行榜
- 适合课程使用
- 教师可在 Firebase Console 查看

### 3. CSV 导出
- 游戏通关后可导出 CSV 文件
- 便于成绩汇总和统计
