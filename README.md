# 组织社会学大富翁 🎮

一个以组织社会学为主题的互动教育游戏，采用类似大富翁的玩法。

## 🎯 游戏特色

- 📚 学习组织社会学的核心概念和理论
- 🎲 投掷骰子，在棋盘上移动
- 💡 回答问题获得积分
- 🎁 随机机会事件
- 🏆 达到100分即可通关

## 🎮 游戏规则

1. **概念格子**（蓝色）：基础概念题，答对 +10 分
2. **理论格子**（橙色）：核心理论题，答对 +20 分
3. **机会格子**（粉色）：随机事件，可能获得或失去 10-15 分
4. 累计 100 分即可完成学习之旅

## 📖 涵盖内容

### 核心概念
- 组织、科层制、组织文化、组织结构
- 权力、官僚主义、组织学习、组织变革
- 组织环境、组织效率、组织创新、组织发展
- 团队合作、决策、组织控制、组织战略

### 重要理论
- 韦伯科层制理论
- 泰勒科学管理
- 法约尔管理原则
- 人际关系学派
- 系统论、权变理论
- 霍桑实验、社会网络理论
- 制度理论、领导力理论
- 激励理论、组织沟通
- 资源依赖理论、利益相关者理论

## 🚀 开始游戏

直接在浏览器中打开 `index.html` 文件即可开始游戏。

## 📄 技术栈

- HTML5
- CSS3 (响应式设计)
- JavaScript (无框架依赖)

## 📜 许可

本项目仅供教育学习使用。
